# **SFML Pong**

This is a pong game built on the SFML library.


## Description

A basic Pong game that support single and multi player mode. Players score based on how many times their paddle successfully hit the ball.
The ball velocity increase as the game progress, increasing the difficulty.
The game provides the ability for user to change background color, volume, and FPS.


## System Requirement

-64-bit CPU (program compiled with 64-bit SFML)
-At least 2 GB system memory
-Git
-Keyboard with WASD and 4 arrow keys
-Display with minimum 800x600 resolution
-Has audio output


## -Installing and Running

	### Windows:

1. Pull this repository
2. Navigate to src\Main
3. Double click "main.exe"

	### MacOS/Linux:

1. Pull this repository
2. Navigate to src/Main
3. Compile "main.cpp" with your favorite C++ compiler
4. Run the executable
